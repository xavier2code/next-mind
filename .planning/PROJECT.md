# Next-Mind AI Agent Framework

## What This Is

Next-Mind是一个面向中型团队（10-50人）的AI Agent协作平台，基于pi-mono框架构建，支持国产大模型（Qwen3.5、GLM、MinMax）。平台提供Web界面和API两种使用方式，集成完整的MCP协议、A2A多Agent协作、Skills技能系统，具备全面的文件处理和RAG知识检索能力，旨在成为团队日常工作的核心AI助手。

## Core Value

让团队成员通过统一的对话界面，高效处理文件、管理知识、调用工具，完成80%以上的日常工作任务。

## Context

**技术背景：**
- 基于pi-mono框架的pi-ai和pi-agent-core组件构建
- pi-mono选择原因：轻量级、支持国产模型、Agent架构清晰
- 需要支持团队协作场景下的多种AI能力整合

**用户痛点：**
- 文件处理：现有工具对多种文件类型支持不好，处理效率低
- 知识管理：团队知识分散，缺乏有效的整合和检索手段
- 工具集成：多个AI工具和服务割裂，缺乏统一入口
- 协作交互：缺乏统一的交互界面供团队使用

**使用场景：**
- 团队成员日常通过对话界面处理文件、查询知识、调用工具
- 支持文件上传后自动识别、分类、提取关键信息
- 通过RAG检索团队知识库、历史对话、外部资源
- 多Agent协作处理复杂任务
- 自定义和编排技能满足特定需求

## Constraints

- **部署环境**: 公有云环境部署 — 团队协作需要外网访问
- **存储方式**: 云存储 — 便于扩展和访问
- **安全合规**: 需要认证授权、数据加密、审计日志、内容安全 — 企业使用要求
- **模型选择**: 必须支持国产大模型 — 成本和合规考虑
- **界面风格**: 对话式UI — 用户熟悉ChatGPT交互模式

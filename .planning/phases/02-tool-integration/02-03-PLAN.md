    Create src/lib/mcp/validation.ts with:

    1. DANGEROUS_PATTERNS constant array:
       - /\$\(/ - Command substitution
       - /`/ - Backtick command substitution
       - /\$\{/ - Variable expansion
       - /\|/ - Pipe (unless explicitly needed)
       - /;/ - Command separator
       - /&&/ - Logical AND
       - /\|\|/ - Logical OR
       - />/ - Output redirect
       - /</ - Input redirect
       - /\n/ - Newline injection

    2. sanitizeCommandArg(arg: string): string
       - Check each DANGEROUS_PATTERNS
       - If pattern found, throw Error with pattern description
       - Return sanitized arg (or throw if unsanitizable)

    3. validatePath(path: string, allowedDirs: string[]): string
       - Reject paths containing "../" or "~/" (path traversal)
       - Resolve to absolute path
       - Check if resolved path starts with any allowedDir
       - Return validated absolute path or throw Error

    4. validateCommand(command: string, allowedCommands: readonly string[]): boolean
       - Check if command is in allowedCommands array
       - Return true if allowed, false otherwise

    Export:
       - DANGEROUS_PATTERNS
       - sanitizeCommandArg
       - validatePath
       - validateCommand
  </action>

    Create src/lib/mcp/tools/bash.ts with:

    1. ALLOWED_COMMANDS constant:
       - Read-only: 'ls', 'cat', 'head', 'tail', 'grep', 'find', 'wc', 'echo', 'pwd'
       - Write (require approval in future): 'mkdir', 'touch', 'rm', 'cp', 'mv'
       - NO: 'sudo', 'su', 'chmod', 'chown', 'dd', 'mkfs', 'fdisk', 'curl', 'wget', 'nc', 'bash', 'sh', 'python', 'node'

    2. DEFAULT_TIMEOUT_MS = 30000
    3. MAX_ARGS = 20

    4. InputSchema using Zod:
       - command: z.enum(ALLOWED_COMMANDS)
       - args: z.array(z.string()).max(MAX_ARGS)
       - timeout: z.number().min(1000).max(60000).default(DEFAULT_TIMEOUT_MS)
       - cwd: z.string().optional()

    5. executeBash(input: unknown, context: { userId: string; sessionId: string }): Promise<McpToolCallResult>
       - Validate input against InputSchema
       - Validate command is in ALLOWED_COMMANDS (already done by Zod enum)
       - Sanitize each arg with sanitizeCommandArg()
       - Check process is not running as root (uid !== 0)
       - Execute using execFile (NOT exec) with:
         - timeout from input
         - cwd: cwd || process.cwd()
         - env: minimal env (PATH, HOME, USER only)
         - uid/gid: from process.env.MCP_UID/MCP_GID or current user
       - Log execution to audit with action: 'bash_execution'
       - Return { content: [{ type: 'text', text: stdout || stderr }], isError: !!error }

    6. Export bashTool: McpTool object ready for registration

    Export:
       - executeBash
       - bashTool
       - ALLOWED_COMMANDS

         - Call resource.read()
         - Return formatted content (text or base64 blob)

    3. Built-in resources:
       - "data://session/info": Returns current session metadata (userId, createdAt)
       - "data://tools/list": Returns list of available tools

       - resourceManager: ResourceManager (singleton for built-in resources)
       - createResourceManager(): ResourceManager (for session-scoped)
